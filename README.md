# 🏭 Task 1: Machine Telemetry Analysis

## 🎯 Objective

Analyze telemetry data collected from four Daikibo factories to answer the following business questions:

1. Which factory experienced the highest machine downtime?
2. Which machine types broke down most frequently in that factory?

## 🛠️ Tools Used

- Tableau
- JSON Dataset

## ✅ Tasks Performed

- Imported and analyzed telemetry data in Tableau.
- Created a calculated measure field named **Unhealthy**.
- Built a **Down Time per Factory** bar chart.
- Built a **Down Time per Device Type** bar chart.
- Created an interactive dashboard with filtering functionality.
- Identified the factory with the highest downtime.

# ⚖️ Task 2: Gender Pay Equality Analysis

## 🎯 Objective

Investigate gender pay equality across different factories and job roles.

## 🛠️ Tools Used

- Microsoft Excel

## ✅ Tasks Performed

- Analyzed employee compensation data.
- Created a new **Equality Class** column.
- Classified equality scores into three categories:

| Equality Score | Classification |
|---------------|----------------|
| -10 to +10 | Fair |
| -20 to -11 and +11 to +20 | Unfair |
| Less than -20 or Greater than +20 | Highly Discriminative |

- Interpreted results to identify inequality patterns.

## 💡 Skills Applied

- Data Analysis
- Data Classification
- Data Interpretation
- Business Intelligence
- Critical Thinking
